Determinism contractWhat “replay safe” means in a browser

The seed, event plan, DSP constants, summation order, quantizer, and WAV writer are explicit. Immediate replays in this app produce identical bytes. JavaScript engines may implement transcendental functions differently, so the receipt records a browser-specific ABI and does not promise byte parity with the Python/NumPy renderer or every future browser.

Playback resampling, device output, filenames, and interface state are excluded from artifact identity.

Scientific boundaryAudible mappings, not physical claims

This is a mathematical composition instrument. Phase 1 uses explicit musical patterns and procedural DSP. It does not claim that E8, quantum computation, or any mapped geometry is physically present in the sound or in nature. More advanced mappings require their own versioned contracts.

Offline guaranteeInspectable by construction

The app uses local classic scripts, bundled hashing, system fonts, Blob downloads, and the Web Audio API. It contains no fetch calls, analytics, service worker, CDN resource, upload path, or account dependency.
